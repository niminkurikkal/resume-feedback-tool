# 📄 Resume Feedback Tool

An AI-powered resume analyser that helps freshers optimise their resumes for placements.

## 🚀 Live App
 [nimin-resume-tool.streamlit.app]https://nimin-resume-feedback-tool.streamlit.app/

##  Features
- Upload your PDF resume and paste a job description
- Get an ATS match score out of 100
- View matched and missing keywords
- AI-generated strengths, weaknesses, and improvement advice
- Persistent score history chart powered by Supabase

##  Tech Stack
- **Python** — core language
- **Streamlit** — web UI framework
- **Groq API (LLaMA 3.3 70B)** — AI engine
- **PyMuPDF** — PDF text extraction
- **Supabase (PostgreSQL)** — persistent database
- **Git + GitHub** — version control
- **Streamlit Cloud** — deployment
